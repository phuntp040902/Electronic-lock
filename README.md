# Description
The electronic security lock uses either a RFID card or a password entered via a keyboard to control the door lock mechanism using a servo module.

1. Proteus
  -  schematic btl.pdsprj:  schematic to stimulate
2. Altium Designer

  -  PCB1.PcbDoc: pcb
3. Arrduino IDE
